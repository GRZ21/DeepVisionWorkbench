# 📈 CSV Result Analyzer

![License](https://img.shields.io/badge/License-MIT-blue.svg)
![C++](https://img.shields.io/badge/Language-C++-orange.svg)
![Qt](https://img.shields.io/badge/Framework-Qt-green.svg)

**CSV Result Analyzer** 是一款专为计算机视觉研究和深度学习（如 YOLO 系列）模型训练日志设计的轻量级、高性能 CSV 数据可视化与分析工具。

它可以帮助研究人员告别繁琐的 Python 画图脚本，通过极其直观的图形界面，一键导入、对比和分析多组实验的 Loss 与 mAP 曲线，并支持一键导出符合学术论文标准的高清插图。

---

## ✨ 核心特性 (Key Features)

* 📂 **批量导入与解析**
  * 支持一键导入整个 `runs` 文件夹，自动递归解析所有子目录下的 `.csv` 训练日志。
  * 智能过滤冗余参数，自动提取核心指标（如 `mAP50`、`box_loss` 等）。
* 🎨 **极度自由的曲线定制**
  * **基础样式**：自定义曲线颜色、透明度、线宽、线型（实线/虚线）。
  * **散点样式**：支持多种散点形状（圆、方、三角、菱形）及大小调节。
  * **图层管理**：支持单条曲线的一键“置顶”与“置底”，解决多曲线遮挡问题。
* 📉 **专业的数据平滑算法(暂未实现）**
  * 内置 EMA（指数移动平均）平滑算法，一键消除震荡，清晰呈现模型收敛趋势。
* 🔄 **全方位状态同步与联动**
  * **图表联动**：点击底部数据表中的某列，自动在左侧树中定位，并在图表中高亮对应的曲线。
  * **双向改名**：右侧面板修改图例名称，左侧文件树节点瞬间同步。
  * **智能高亮**：数据表、文件树、图表与右侧属性面板状态 100% 实时同步。
* 🎓 **学术级图表导出**
  * 支持自定义背景网格（论文标准虚线）、全局字体（如 Arial）、坐标轴标题。
  * 支持按固定比例或当前窗口比例，一键导出高清图表，直接用于学术论文插图。
* 🖥️ **沉浸式专业 UI 布局**
  * 采用经典的“左侧文件树 - 中间图表与数据 - 右侧属性面板”布局。
  * 自适应拉伸，支持各个面板的独立显示/隐藏，随时进入纯净看图模式。

### 布局说明
1. **左侧（数据源区）**：清晰的树状结构，管理多组实验文件及内部指标，支持右键移除文件。
2. **中间（核心工作区）**：
   * **上半部**：基于 `QCustomPlot` 驱动的高性能绘图区，支持滚轮缩放、鼠标拖拽。
   * **下半部**：完整的原始 CSV 数据表格，方便查阅具体数值。
3. **右侧（属性控制区）**：
   * **曲线属性**：精准控制当前选中曲线的外观、平滑度，并提供最大/最小/最终值的统计信息。
   * **全局设置**：管理图表整体的网格、图例、字体和导出。
